# 🚀 Codmint Technologies: Internship Platform

> **"Where Developers Begin"**

Welcome to the Codmint Technologies Internship Platform. Our mission is to provide a seamless, inspiring, and professional gateway for emerging developers to kickstart their careers. This platform is designed not just to function flawlessly, but to feel effortless—utilizing a custom "anti-gravity" design system.

---

## ✨ The "Anti-Gravity" Experience

We believe a developer's first step should feel weightless. Our UI/UX is built on the principles of **anti-gravity design**:
- **Clean Aesthetic**: A minimalist corporate look using a pristine white background with striking Red (`#FF2C2C`) and Dark Grey (`#353839`) accents.
- **Weightless Interactions**: Cards and interactive elements feature soft, large-radius shadows and subtle Y-axis hover offsets (e.g., `-8px` lift).
- **Dynamic Flow**: Smooth `fade-in-up` scroll reveals and a semi-transparent, glassmorphism-inspired sticky navigation bar ensure the interface feels alive and responsive.

## 🛠️ Technical Stack

This project is built with a focus on core web technologies to ensure maximum performance, complete styling control, and a responsive layout without relying on heavy external frameworks.

* **Markup & Logic**: HTML5, Vanilla JavaScript (ES6+)
* **Styling**: Vanilla CSS3 (Custom Design System, Flexbox/Grid)
* **Typography**: Poppins (Google Fonts)
* **Version Control**: Git & GitHub (Adhering to Conventional Commits)

## 📋 Key Features

- **Internship Domain Explorer**: Browse various tech domains presented in floating, interactive cards.
- **Certificate Verification Portal**: A dedicated, easy-to-use section for verifying internship completion certificates.
- **Fully Responsive**: Optimized for Desktop, Tablet, and Mobile devices.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps:

### Prerequisites
* A modern web browser (Chrome, Firefox, Safari, Edge).

### Installation

1. **Clone the repository**
   ```sh
   git clone https://github.com/your-username/codmint-internship-platform.git
   ```
2. **Navigate to the project directory**
   ```sh
   cd codmint-internship-platform
   ```
3. **Launch the project**
   Simply open the `index.html` file in your preferred web browser.
